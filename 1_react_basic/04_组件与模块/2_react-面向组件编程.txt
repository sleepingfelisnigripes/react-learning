react面向组件编程有两种类型
    1. 函数式组件：适用于简单组件
        简单组件： 如果组件没有state状态，就是简单组件
    2. 类式组件： 适用于复杂组件
        复杂组件： 如果组件是有状态state的，就是复杂组件

什么是state？
    人 -> 状态 -> 影响 -> 行为
    组件 -> 状态 -> 驱动 -> 页面
    数据是放在state里面，组件的状态里面存着数据，数据的改变就会驱动着页面的展示。
    (4-4调控制台看MyComponent看组件对象有主要的四个部分：
        context:{}
        props:{}
        refs:{}
        state:null 这个就是React为我们准备好的状态，这里面的值等着我们去具体赋值。
    )
    state是组件实例对象生成的，而不是组件类本身生成的

注意，组件的核心属性只有类式组件才有，因为class才有实例化，function没有
但是最新版本function也可以用hooks有这三大属性
